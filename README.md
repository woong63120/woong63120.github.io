# Woong's Portfolio & Tetris Game

This project contains my personal portfolio landing page and a custom-built 2D Tetris game.

## File Structure

```
woong63120.github.io/
├── index.html        # Main Landing Page (New)
├── style.css         # Global Styles (New)
├── main.js           # Global Logic (New)
├── README.md         # Documentation
└── tetris/           # Tetris Game Directory (Moved)
    ├── index.html    # Game Page
    ├── style.css     # Game Styles
    ├── main.js       # Game Logic
    └── *.jpg         # Game Assets
```

## Features

-   **Portfolio**: Modern, responsive landing page with glassmorphism design.
-   **Tetris**: A fully functional 2D Tetris game with scoring, levels, particle effects, and sound placeholders.

## How to Run

1.  Open `index.html` in your browser.
2.  Navigate to the Tetris game via the "Featured Projects" section.

## Technologies

-   HTML5
-   CSS3 (Custom properties, Flexbox/Grid)
-   JavaScript (ES6+, Canvas API)
